hexo关联github搭建免费博客

发表于 2019-01-21
第一步：创建github仓库及域名
1.1 注册github账号

​ 如果已有账号，请直接跳过。打开github注册地址，按照提示依次输入用户名、邮箱、密码进行注册。需要注意的是，在用户注册账号的同时，github会默认给用户分发一个二级域名，如果你的用户名为username，那么你的域名就为username.github.io，也就是说，你的用户名会影响到你的域名。

1.2 创建仓库

​ 登陆github之后，点击右上角的 ‘+’号，点击new repository按钮创建一个新仓库。仓库名为username.github.io，然后点击create repository按钮进行创建。

第二步： 安装软件
2.1 安装node环境

​ 到nodejs中文网下载对应的安装包，这里建议下载.msi版本，因为该版本不需要自己配置，傻瓜式安装即可。安装完成之后，win+R输入cmd运行，在命令行输入node -v、npm -v显示对应的版本号，则表明node安装成功。

2.2 安装Git命令行工具

​ 到Git官网下载对应的安装包，按照提示一步步完成安装。在cmd命令行输入git --version能得到对应的版本号，则表明git安装成功。

2.3 安装hexo 框架

​ 至此，node以及git已经安装完成，我们还需要安装hexo。鼠标右键点击Git Bash Here打开git命令行工具，执行hexo安装命令，等待安装。安装完成之后，输入hexo -v显示对应版本号，表明hexo安装成功。

第三步： 创建本地博客
3.1 初始化项目

​ 在你希望存放博客目录的文件夹下右键点击Git Bash Here，输入hexo init blogname，执行成功后，会生成一个名为blogname的文件夹。

3.2 测试

​ 切换至blogname文件夹下，执行命令hexo s启动服务，然后在浏览器中输入localhost:4000就可以访问你刚搭建好的本地博客。

3.3 安装主题

​ 为了让博客更具个性化，我们需要安装主题 ，Hexo提供了丰富的主题供你选择。这里我们以Next主题为例进行安装。

​ 稍等片刻，你会看到blogname文件夹下的themes文件夹下会生成一个Next文件夹，表明该主题已经下载完成。

3.4 配置博客

​ 打开文件blogname\_config.yml，修改其中的一些键值对。注意键和值之间必须有空格。

​ 修改完成之后，保存退出。重新启动hexo 服务，你会看到你的主题已经更改完成。

​ 配置github的ssh密钥可以让本地git项目与远程的github建立联系，这样我们在本地创建代码之后就可以直接通过git操作，将本地项目同步至Github代码库。
